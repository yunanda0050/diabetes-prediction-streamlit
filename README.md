# 🩺 Diabetes Risk Prediction Web Application using Machine Learning

This project is an interactive web application designed to predict the risk of diabetes in patients based on key health indicators. The entire development process of this project is strictly guided by the **CRISP-DM** *(Cross-Industry Standard Process for Data Mining)* methodology.

## 📈 Business Understanding

### Short Description
This project aims to predict diabetes using machine learning with a CRISP-DM approach to detect disease risks at an early stage. By analyzing health factors such as blood glucose levels, blood pressure, and Body Mass Index (BMI), this model is expected to assist medical professionals in making faster and more accurate clinical decisions.

The CRISP-DM framework—which encompasses six main phases: Business Understanding, Data Understanding, Data Preparation, Modeling, Evaluation, and Deployment—ensures that the analytical process is conducted systematically and effectively. The outcome of this project is expected to contribute to increasing public health awareness and supporting the development of healthcare technology driven by Artificial Intelligence.

### ❓ Problem Statements
1. How can we build a machine learning model capable of predicting early-stage diabetes risks with a high level of accuracy?
2. Which health factors (such as blood glucose levels, blood pressure, and BMI) exert the highest influence on increasing diabetes risks based on the data analysis?
3. How can the implementation of the six CRISP-DM phases ensure that the entire analytical process runs systematically and effectively?

### 🎯 Objectives
* To build a machine learning model using the CRISP-DM approach for early-stage diabetes risk prediction.
* To deliver an accurate and reliable model that helps medical professionals make clinical decisions faster and more precisely.
* To raise public awareness regarding the importance of early detection and to support the advancement of AI-driven healthcare technology.

## 📊 Data Understanding & Data Description
The data utilized in this study is sourced from the dataset titled `diabetes_prediction_dataset.csv`, which contains **100,000 rows of data with 9 attributes**.

This dataset consists of 8 independent attributes covering various risk factors (including gender, age, hypertension history, blood glucose levels, and BMI) and 1 target attribute (class) indicating whether a person is at risk of diabetes or not.
